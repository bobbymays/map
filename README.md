# map
How to install

1. Right click on your FiveM Icon and select "Open file location"
2. Open FiveM Application Data
3. Nagivate to citizen\common\data\ui and place "mapzoomdata.meta" in the "ui" folder
4. Done

If you still need help, install video: https://www.youtube.com/watch?v=InWcKemhZVU

If you still see a grey map, contact staff.
